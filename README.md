# Outgrow Spreadsheet Handler

[![clasp](https://img.shields.io/badge/built%20with-clasp-4285f4.svg)](https://github.com/google/clasp)

This is a Javascript copy of the code base at Google Apps Script. To use

- Copy all files to an app scrip project binded to your instance of the "Outgrow - Database". This can be done by will the "Extension" menu Item then clicking on App Script.
- When moving the files, all *.js file should be renamed to *.gs
- The Script will create a menu item('Generate Result') that can be used as an entry point
